# FinTrack

FinTrack is a simple web app for tracking your expenses, budgets, and categories. It helps you manage your money and see reports easily.

## Features
- Add and view expenses
- Set budgets for categories
- Organize expenses by categories
- See dashboard summary and charts
- View detailed reports
- Switch between dark and light mode

## Project Structure
- `index.html` - Main page
- `dashboard.html` - Dashboard summary
- `expenses.html` - Manage expenses
- `categories.html` - Manage categories
- `budget.html` - Manage budgets
- `report.html` - View reports
- `js/` - All JavaScript files
  - `app.js` - App entry point
  - `common.js` - Shared functions (theme, notifications)
  - `data.js` - Data definitions and storage
  - `dashboard.js` - Dashboard logic
  - `expenses.js` - Expense logic
  - `categories.js` - Category logic
  - `budget.js` - Budget logic
  - `report.js` - Report logic
- `css/style.css` - Styles
- `assets/images/` - Images

## How it works
1. Data is stored in your browser (localStorage).
2. You can add, edit, and delete expenses, categories, and budgets.
3. The dashboard and report pages show charts and summaries.
4. You can switch themes using the toggle button.

## How to run
Just open `index.html` in your browser. No server needed.
